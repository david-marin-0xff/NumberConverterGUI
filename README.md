\# NumberConverterGUI



NumberConverterGUI is a Windows app that allows you to convert numbers between \*\*Decimal, Binary, Hexadecimal, and Octal\*\*. It also provides explanations of the C# conversion process.



\## Features



\- Convert numbers between Decimal, Binary, Hex, and Octal

\- Shows explanation of the conversion using C# formulas

\- Red-themed, resizable GUI

\- Fully portable Windows executable (no .NET installation required)



\## How to Run



1\. Download the latest release from the \[Releases page](https://github.com/david-marin-0xff/NumberConverterGUI/releases).

2\. Extract the files if needed.

3\. Double-click `NumberConverterGUI.exe` to run the application.
